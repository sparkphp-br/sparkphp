# AI SDK

O SparkPHP agora inclui uma SDK de AI unificada com uma unica entrada:

```php
ai()
```

Em vez de separar texto, embeddings, imagem, audio e agentes em APIs desconectadas,
o Spark oferece uma DX coerente e provider-agnostic desde o inicio.

## Filosofia

- uma entrada unica: `ai()`
- builders pequenos e previsiveis
- driver fake por default para desenvolvimento e testes
- adaptadores provider-agnostic via `AiProvider`
- mesma linguagem do resto do framework: pouco boilerplate, sem registro verboso

## Configuracao

No `.env`:

```env
AI_DRIVER=fake
AI_TEXT_MODEL=spark-text
AI_EMBEDDING_MODEL=spark-embedding
AI_IMAGE_MODEL=spark-image
AI_IMAGE_SIZE=1024x1024
AI_AUDIO_MODEL=spark-audio
AI_AUDIO_VOICE=default
AI_AUDIO_FORMAT=mp3
AI_AGENT_MODEL=spark-agent
```

Hoje o Spark shipa com o driver `fake`, pensado para:

- smoke tests
- desenvolvimento local
- contratos previsiveis na suite

Drivers reais podem ser registrados sem mudar a API da aplicacao.

## Texto

```php
$response = ai()
    ->text('Resuma a proposta comercial em 3 bullets.')
    ->system('Seja objetivo.')
    ->temperature(0.2)
    ->generate();

return (string) $response;
```

## Embeddings

```php
$embeddings = ai()
    ->embeddings([
        'SparkPHP prioriza previsibilidade.',
        'Laravel prioriza ecossistema.',
    ])
    ->generate();

$firstVector = $embeddings->first();
```

## Retrieval

O mesmo cliente agora tambem encurta retrieval:

```php
$documents = ai()->retrieve('Como configuro cache?')
    ->from('documents', 'embedding')
    ->take(3)
    ->get();

$context = $documents->toPromptContext('content');
```

## Imagem

```php
$image = ai()
    ->image('Um painel de analytics laranja e grafite, estilo editorial')
    ->size('1024x1024')
    ->generate();

$binary = (string) $image;
```

## Audio

```php
$audio = ai()
    ->audio('Bem-vindo ao SparkPHP.')
    ->voice('default')
    ->format('mp3')
    ->generate();
```

## Agentes

```php
$agent = ai()
    ->agent('support')
    ->instructions('Responda de forma curta e tecnica.')
    ->tool('lookupOrder', fn(array $args) => ['id' => $args['id'] ?? null], 'Consulta pedido')
    ->context(['team' => 'support'])
    ->prompt('Como encontro o pedido 123?')
    ->run();

return (string) $agent;
```

## Structured output

Texto e agentes tambem podem pedir payload estruturado com a mesma API curta:

```php
$result = ai()->text('Extraia nome e email')
    ->schema([
        'type' => 'object',
        'properties' => [
            'name' => ['type' => 'string'],
            'email' => ['type' => 'string', 'format' => 'email'],
        ],
    ])
    ->generate();

return $result->structured;
```

## Driver fake

O driver fake e o comportamento default da SDK no estado atual do framework.

Exemplo:

```php
$text = ai()->text('Hello')->generate();

echo (string) $text;
// Fake response: Hello
```

Isso permite validar fluxo, contratos e integracao sem depender de rede, custo ou
credenciais externas.

## Registrando um provider custom

```php
$manager = app()->getContainer()->make(AiManager::class);

$manager->extend('custom', function () {
    return new class implements AiProvider {
        public function name(): string { return 'custom'; }

        public function text(AiTextRequest $request): AiTextResponse
        {
            return new AiTextResponse('ok', 'custom', $request->model);
        }

        public function embeddings(AiEmbeddingRequest $request): AiEmbeddingResponse
        {
            return new AiEmbeddingResponse([[1.0, 0.0]], 'custom', $request->model);
        }

        public function image(AiImageRequest $request): AiImageResponse
        {
            return new AiImageResponse('image', 'custom', $request->model);
        }

        public function audio(AiAudioRequest $request): AiAudioResponse
        {
            return new AiAudioResponse('audio', 'custom', $request->model);
        }

        public function agent(AiAgentRequest $request): AiAgentResponse
        {
            return new AiAgentResponse('agent', 'custom', $request->model);
        }
    };
});

$reply = ai('custom')->text('ping')->generate();
```

## Contratos da SDK

As classes principais do core sao:

- `AiManager`
- `AiClient`
- `AiProvider`
- `AiTextRequest`, `AiEmbeddingRequest`, `AiImageRequest`, `AiAudioRequest`, `AiAgentRequest`
- `AiTextResponse`, `AiEmbeddingResponse`, `AiImageResponse`, `AiAudioResponse`, `AiAgentResponse`
- `AiTool`
- `AiFakeProvider`

## Convencoes file-based
